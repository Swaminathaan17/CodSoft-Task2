# 🚀 CodSoft Task 2 - Landing Page

A modern and responsive Landing Page built using HTML and CSS as part of the CodSoft Web Development Internship.

## 📌 Project Overview

This project demonstrates the fundamentals of front-end web development by creating an attractive and responsive landing page. The design focuses on clean layouts, proper alignment, visual appeal, and a seamless user experience.

## ✨ Features

* Responsive Design
* Modern User Interface
* Hero Section
* Navigation Bar
* Call-to-Action Buttons
* Multiple Content Sections
* Footer Section
* Mobile-Friendly Layout

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🎯 Learning Outcomes

* HTML page structuring
* CSS styling and layouts
* Responsive web design
* Flexbox and Grid concepts
* UI/UX design fundamentals
